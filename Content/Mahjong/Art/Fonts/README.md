# 麻将游戏字体资源

本目录包含适合麻将游戏使用的中文字体。

## 字体列表

| 文件名 | 字体名称 | 风格 | 文件大小 |
|--------|----------|------|----------|
| `simkai.ttf` | 楷体 | 楷书风格，适合传统麻将牌面 | 11.2 MB |
| `STXIHEI.TTF` | 黑体 | 简洁现代，适合UI界面 | 9.3 MB |
| `STKAITI.TTF` | 楷体 | 华文楷体，书法风格 | 12.1 MB |

## 麻将字符覆盖

这些字体都完整支持以下麻将字符:

- **萬子**: 一萬 二萬 三萬 四萬 五萬 六萬 七萬 八萬 九萬
- **筒子**: 一筒 二筒 三筒 四筒 五筒 六筒 七筒 八筒 九筒
- **條子**: 一条 二条 三条 四条 五条 六条 七条 八条 九条
- **风牌**: 东风 南风 西风 北风
- **字牌**: 红中 发财 白板
- **花牌**: 梅 兰 菊 竹
- **特殊**: 鸡 金鸡 赖子

## 在UE中使用

1. 打开UE编辑器
2. 在Content Browser中导航到 `Content/Mahjong/Art/Fonts/`
3. 右键点击字体文件 → Asset Actions → Create Font Instance
4. 设置Font Family为对应中文字体

## 建议

| 用途 | 推荐字体 |
|------|----------|
| 牌面文字 | simkai.ttf (楷体) |
| UI标题 | STXIHEI.TTF (黑体) |
| 按钮文字 | STKAITI.TTF (楷体) |

## 运行脚本

```bash
python _generate_fonts.py
```

将生成字体预览图(preview.png)并显示字体信息。

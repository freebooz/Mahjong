# UI按钮资源 (Button Assets)

## 必需按钮
- btn_draw.png - 摸牌按钮
- btn_discard.png - 出牌按钮
- btn_peng.png - 碰按钮
- btn_gang.png - 杠按钮
- btn_hu.png - 胡按钮
- btn_guo.png - 过/取消按钮
- btn ting.png - 报听按钮
- btn_zhuaji.png - 捉鸡按钮

## 推荐规格
- 尺寸: 128x64 (正常状态), 128x64 (按下状态)
- 格式: PNG 带透明通道
- 建议制作Normal/Hover/Pressed/Disabled四种状态

## 设计风格
- 圆角矩形
- 半透明背景
- 麻将风格配色 (绿/金/红)

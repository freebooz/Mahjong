# 动作音效 (Action Sounds)

## 文件列表
- action_peng.wav - 碰牌音效
- action_gang.wav - 杠牌音效
- action_hu.wav - 胡牌音效
- action_chow.wav - 吃牌音效
- action_discard.wav - 出牌音效
- action_ting.wav - 报听音效

## 推荐规格
- 格式: WAV (44.1kHz, 16bit) 或 OGG
- 时长: 0.2s - 1.0s
- 风格: 麻将牌清脆撞击声
